# Sai Ganesh Manikantaraju Nagulapalli | AWS-Focused Developer Portfolio

This is a responsive, animated personal portfolio built with React + Vite. It includes a sticky navigation bar, hero/about/skills/projects/contact sections, scroll-triggered motion, a subtle parallax hero layer, and accessibility-first styling.

## Tech Stack

- React 19
- TypeScript
- Vite
- Framer Motion
- Lucide React
- Custom CSS
- Google Fonts

## Local Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Build for production:

   ```bash
   npm run build
   ```

## Project Notes

- Replace the placeholder contact links and project repository URLs in [src/data/portfolio.ts](src/data/portfolio.ts) with your real information before publishing.
- The site is designed to be tested at mobile, tablet, and desktop widths.
- Reduced-motion support is built into the global stylesheet.
