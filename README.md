# Sai Ganesh Manikantaraju Nagulapalli | AWS-Focused Developer Portfolio

This is a responsive, animated personal portfolio built with React + Vite. It includes a sticky navigation bar, hero/about/skills/projects/contact sections, scroll-triggered motion, a subtle parallax hero layer, and accessibility-first styling.

## Tech Stack

- React 19
- TypeScript
- Vite
- Framer Motion
- Lucide React
- Custom CSS
- Google Fonts

## Deploy to Vercel (Recommended)

1. Go to https://vercel.com
2. Click "New Project"
3. Import the GitHub repository
4. Click "Deploy"
5. Your live URL will be auto-generated (e.g., `personal-portfolio-website.vercel.app`)
6. Update the README with your live URL below

## Deploy to Netlify

1. Go to https://netlify.com
2. Click "New site from Git"
3. Connect GitHub and select this repository
4. Click "Deploy"
5. Your live URL will be auto-generated

## Local Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Build for production:

   ```bash
   npm run build
   ```

## Project Notes

- Replace the placeholder contact links and project repository URLs in [src/data/portfolio.ts](src/data/portfolio.ts) with your real information before publishing.
- The site is designed to be tested at mobile, tablet, and desktop widths.
- Reduced-motion support is built into the global stylesheet.
